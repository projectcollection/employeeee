# simple cli tool

## commands prefix: emp
- all: show all employees
- add: add an employee
- show [id]: show employee by id
- del [id]: delete an employee by id
- reset: delete everything
- h: show this message

## usage

install globally
```
//from project root
npm i -g
```

uninstall
```
npm uninstall -g employeeee
```

show usage options
```
emp
```
